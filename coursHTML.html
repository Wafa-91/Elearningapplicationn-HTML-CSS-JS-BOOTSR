Learn how to create your website with HTML5 and CSS3

Log in or sign up for free to enjoy all this course has to offer!

Do you want to create your own websites? You are in the right place ! In this course, you will learn how to use HTML5 and CSS3, the two programming languages ​​that are the basis of all websites.

You may think that you are not cut out for learning a computer language, but don't worry: HTML and CSS are simple languages ​​that we will take you step by step through many exercises. You will soon be able to add text to your site, build a navigation menu, insert images ... and much more!

So, ready to build a website from A to Z?

Educational goals

By the end of this course, you will be able to:

  • use HTML code;
  • structure a web page in HTML;
  • use CSS code;
  • format a web page in CSS;
  • organize the elements of a web page using CSS;
  • modify the layout of an HTML page with CSS;
